1.First, you need to download Eclipse, which is a program that helps you write and run Java code. Once it's downloaded, you'll install it like any other program on your computer.
2.After installation, find the Eclipse icon on your desktop.
3.When you open Eclipse for the first time, it'll ask you to choose a place to store all your projects. This is called your "workspace." You can pick the default location or choose your own.
4.Once you're in Eclipse, you'll want to create a new Java project. You do this by going to the "File" menu at the top, then selecting "New" and "Java Project."
5.Give your project a name, Java_Development_Assignment, and hit "Finish." Eclipse will set up a new project for you.
6.Now it's time to write some Java code! In Eclipse, you'll see a panel on the left called the "Package Explorer." This shows all your projects and files. Find your new project and right-click on the "src" folder.
7.From the menu that pops up, choose "New" and then "package." and give the name Q1 and Finish. Find your Q1 PACKAGE and right-click on it.
8.From the menu that pops up, choose "New" and then "Class." This will create a new Java class file for you to write your code in. Give the class a name, like "numeric array sorter". This creates a special method in your class that Java knows to run when you start your program.
8.Now, you can start writing your Java code inside the main method you just created. This is where your program will start running.
9.Once you've written your code, it's time to see it in action! Right-click on the Java file where you wrote your main method  in the Package Explorer.
10.From the menu that pops up, choose "Run As" and then "Java Application." Eclipse will run your program, and you should see the output in the Console view at the bottom of the Eclipse window.
